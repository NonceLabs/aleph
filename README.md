# Aleph Reader

> Aleph Reader is a client for RSS and podcast. It is built with React-Native.

## Features

- [x] RSS
- [x] Podcast
- [x] Dark Mode
- [x] Summary
- [x] Tags
- [x] Feedback to openAI
